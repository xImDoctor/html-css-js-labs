Задание 1

Создайте переменные age и gender, запросите у пользователя их значение. Используя конструкцию if ... else напишите скрипт, который проверяет значение переменных age, gender и:

  • если значение age попадает в диапазон 0–17 (вкл.) выводит сообщение "Вам работать ещё рано — учитесь"
  • если значение age попадает в диапазон 18–59 (вкл.) для мужчин или в диапазон 18–54 (вкл.) для женщин выводит сообщение "Вам ещё работать и работать"
  • если значение age попадает в диапазон 60–64 (вкл.) для мужчин или в диапазон 55–59 (вкл.) для женщин выводит сообщение "Скоро пенсия!"
  • если значение больше 65 для мужчин или больше 60 для женщин выводит сообщение "Вам пора на пенсию"
  • в остальных случаях выводитсясообщение "Да кто ты такой?"
